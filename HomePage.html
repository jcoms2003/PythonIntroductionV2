To start coding you need to understand variables and data types. See Data Types and Variables for more information. Variables are values that have different types, for example a whole number is an integer or int and decimals are float. The program will automatically assign the type based on the initial value, however, you have the option to assign it yourself depending on the situation. Variables can also be words; these are called strings. They can also be stored in a variable as a str (). Next you have Booleans, which are basically true or false (1/0). Finally, with variables you have the ability to use operators, or in other words basic arithmetic (* / + / -).
See w3schools for more information.

Now, getting into the core of python we'll talk about lists, tuples, sets, dictionaries, and conditionals.

  • Lists are multiple items stored into a single variable. They are changeable, ordered, and allow duplicates.
  • Tuples are the same as lists except they are unchangeable.
  • Sets are also the same as lists except they are unordered, unchangeable and unindexed.
  • Dictionaries are used to store data in key:value pairs

Finally...

Moving on, we have conditionals. Conditionals are exactly as they sound. They are parameters that must be met in order to complete an action. To see the syntax and understand conditionals more visit Learning Conditionals with Python.
