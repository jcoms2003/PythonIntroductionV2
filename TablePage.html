Guide for Python Syntax

Keyword Description Code Example
True, False Boolean Data Types False == (1 > 2), True == (2 > 1)
print(" ") Prints argument in parentheses onto the console print("hello world")
if, elif, else Conditional program execution: program starts with “if” branch, tries the “elif” branches, and finishes with “else” branch (until one branch evaluates to True). 
    x = int(input("your value: "))
    if x > 3: print("Big")
    elif x == 3: print("Medium")
    else: print("Small")
lambda Function with no name (anonymous function) (lambda x: x + 3)(3) # returns 6
def() Used to create a custom function 
    def My_Function(x):  # function name
        print(x**2) # Prints the square root of the number "x"
    My_Function(5)  # Prints 25 in the console
return Terminates execution of the function and passes the flow of execution to the caller. An optional value after the return keyword specifies the function result. 
    def incrementor(x):
        return x + 1
    incrementor(4) # returns 5
in Checks whether element is in sequence 42 in [2, 39, 42] # True
